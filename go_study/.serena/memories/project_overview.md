Go-Study is a Go learning/practice project covering context, DI, repository pattern, transactions, Kafka, OTEL tracing/logging, HTTP API, and CLI. Module: github.com/bruno303/study-topics/go-study. Main entrypoints: cmd/api/main.go and cmd/cli/main.go. Core layers: internal/application (use-cases/contracts), internal/infra (adapters: api, repository, kafka, db), internal/crosscutting (observability), internal/setup (container wiring), tests/integration (integration helpers), pkg (utilities). Uses PostgreSQL (pgx), Kafka (confluent client), OpenTelemetry, YAML/env config.
