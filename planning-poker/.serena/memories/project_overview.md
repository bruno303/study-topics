Planning Poker is a full-stack agile estimation app. Backend is Go (clean-ish layered architecture: domain -> application -> infra) and frontend is Next.js 15 + React 19 with strict TypeScript in frontend/planning-poker-front. Core goal is real-time collaborative estimation via WebSockets, with Redis used for coordination/pubsub and scaling. Main backend entrypoint is cmd/api/main.go; frontend app lives under frontend/planning-poker-front/src.
